Les problèmes rencontrés et leur résolution

(Cette partie va être écrite en anglais pour une explication plus claire.)

npm install (Référence[2])

Error: Cannot find module ‘internal/util/types’

The reason seems to be the compatibility of nodeJS module library. I reinstalled nodeJS and fixed it by getting an older version of nodeJS.

step:

  1. remove nodeJS (8.9.3)
  2. reinstall nodeJS(6.2.2)
  3. update npm (command: npm update)
  4. restart npm install

We will need to have in mind that we need an older version during the initialization that’s why I restarted ‘node-v8.9.3’ to update nodeJS when npm install had finished.

npm run init

Error: Cannot find module ‘fs-extra’

I solved it by npm install fs-extra and update again nodeJS version.

MongoDB connection error

MongoDB connection error: MongoError: connect ECONNREFUSED 127.0.0.1:27017

I forgot to connect mongoDB. Once it was connected, the problem was solved.

Webpage loading for a long time but not loaded.

It appears when the server is connected but the client is down, solved simply by connecting the client.
